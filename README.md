## 📜 How It Works

### 🔁 On Page Load:
- The entire page fades in using CSS `@keyframes`.

### 👇 On Scroll:
- Each `.fade-on-scroll` section becomes visible with a smooth `translateY + opacity` animation.
- Powered by **IntersectionObserver API** in `main.js`.
